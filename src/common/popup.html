Help for DigitalOcean Toolkit

Create DigitalOcean acount

To use DigitalOcean Toolkit browser extension you need a DigitalOcean account. If you already have an account, you can directly go to next step "Prepare Toolkit...".

  1. Create your own DigitalOcean account.
  2. Verify your new created account.

Get your own DigitalOcean account here.

Prepare Toolkit for using DigitalOcean API

To use DigitalOcean Toolkit browser extension you need a personal access token.

  1. Create your personal access token for example named as "Toolkit".
  2. Copy "Toolkit" personal access token.
  3. Open option dialog of DigitalOcean Toolkit browser extension.
  4. Paste copied personal access token into token field.
  5. Save your settings.

Get your personal access token for DigitalOcean API here.

Info about DigitalOcean Toolkit

With the DigitalOcean Toolkit browser extension it is currently possible to manage your droplets via APIv2 of DigitalOcean.

Support development of DigitalOcean Toolkit

If you like this little browser extension and want more features you can support in different ways.

  • Leave a positive rating or review in browser extension store.
  • Share this extension with other DigitalOcean users you know.
  • Blog about DigitalOcean Toolkit browser extension on your website.
  • Use paypal.me to donate a small amount for further development of DigitalOcean Toolkit.
  • Use DigitalOcean referral link to create new DigitalOcean accounts.
