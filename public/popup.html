About DigitalOcean Toolkit

With the DigitalOcean Toolkit browser extension it is currently possible to manage your droplets via APIv2 of DigitalOcean.

Initial development

Initial development was done by github user jasonswan under project name digitalocean-toolbox. The old DigitalOcean Toolbox only supports APIv1.

Support DigitalOcean Toolkit

If you like this little browser extension and want more features you can support in different ways.

  • Use DigitalOcean referral link to create new DigitalOcean accounts.
  • Use paypal.me to donate a small amount for further development of DigitalOcean Toolkit.
  • Leave a positive rating or review in browser extension store.
  • Share this extension with other DigitalOcean users you know.
  • Blog about DigitalOcean Toolkit browser extension on your website.
